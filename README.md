# 🎓 Student Smart Portal System

A modern university management system built using React, TypeScript, and Tailwind CSS.

The platform provides separate dashboards and functionalities for Students, Teachers, and Admins with a clean responsive UI and role-based access system.

---

# 🚀 Features

## 👨‍🎓 Student Dashboard
- View registered courses
- Access academic information
- Modern responsive interface
- Easy navigation system

## 👨‍🏫 Teacher Dashboard
- Manage courses and students
- View assigned subjects
- Organized dashboard layout

## 👨‍💼 Admin Dashboard
- Full system management
- Manage students, teachers, and courses
- Administrative controls and monitoring

---

# 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- CSS3
- HTML5
